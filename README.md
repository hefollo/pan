# 🌈 彩虹外链网盘

基于彩虹外链云盘二开 + 美化

![项目预览](https://mpimg.cn/view.php/2841008ec06283dcd8378d7045992dcf.png))

## 项目介绍

彩虹外链网盘是一款基于 PHP 开发的网盘与外链分享程序，支持多种类型文件上传，并可生成对应的外链地址，适用于文件分享、图床、音视频外链等多种使用场景。

本项目是在开源版本基础上进行二次开发与界面美化的版本，保留了原有核心功能，并对前后台主题风格、交互体验和部分功能展示进行了优化。

## 主要功能

- 支持所有常见格式文件上传
- 支持生成文件外链、图片外链、音乐外链、视频外链
- 生成外链时可同时生成对应的 UBB 代码和 HTML 代码
- 支持文本、图片、音乐、视频在线预览
- 支持 TXT 等文本内容在线查看
- 支持文件内容在线编辑
- 在线编辑支持以下格式：
  `txt`、`text`、`log`、`md`、`markdown`、`json`、`js`、`mjs`、`css`、`less`、`scss`、`html`、`htm`、`xml`、`svg`、`csv`、`ini`、`conf`、`config`、`yaml`、`yml`、`sql`、`vue`、`ts`、`tsx`、`jsx`
- 可作为网盘使用，也可作为图床或音乐在线试听网站使用
- 支持对接多种云存储服务
- 支持图片违规检测功能

## 支持的云存储

新版本支持对接以下云存储平台：

- 阿里云 OSS
- 腾讯云 COS
- 华为云 OBS
- 又拍云
- 七牛云

## 项目说明

本项目为基于开源程序的二开美化版本，主要包含：

- 前台界面样式优化
- 后台管理界面美化
- 外观主题扩展
- 新增外观设置功能，可在后台切换多套前后台联动主题风格
- 新增广告公告位设置功能，可在后台统一管理顶部公告、广告位内容及展示方式
- 新增在线查看文本内容功能
- 新增在线编辑功能
- 在线编辑白名单已扩展，适合文本、前端代码、配置文件和部分脚本文件直接在线修改
- 部分功能体验调整与增强

## 二开与新增内容

相较于原版程序，本项目额外新增或强化了以下内容：

- 外观设置：
  后台新增外观设置页面，可切换不同站点主题，并同步作用于前台页面与后台管理界面，不再只是单独修改前台样式。

- 广告公告位设置：
  后台新增广告公告位设置页面，可自定义公告内容、广告位内容、广告类型、展示开关等，方便统一维护站点顶部公告与首页广告位。

- 在线查看与在线编辑：
  新增文本内容在线查看功能，并支持多种文本类、前端类、配置类文件在线编辑，方便直接在网页中快速处理文件内容。

- 界面美化：
  对前台文件列表、上传页、文件查看页，以及后台管理界面进行了整体风格优化与主题适配增强。
